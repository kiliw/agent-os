# Business-Prompts

Prompts für strategische Analyse und Planung. Diese Prompts helfen dir, fundierte Entscheidungen schneller zu treffen – durch strukturierte Analyse statt vager Einschätzungen.

## Verfügbare Prompts

| Prompt | Zweck | Dauer |
|--------|-------|-------|
| [`market-analysis.md`](./market-analysis.md) | Vollständige Marktanalyse für eine Branche/Nische | 10–20 Min |
| [`business-model-canvas.md`](./business-model-canvas.md) | Business Model Canvas aus einer Idee | 5–10 Min |
| [`competitor-research.md`](./competitor-research.md) | Wettbewerbsanalyse mit Stärken/Schwächen/Lücken | 10–15 Min |

## Wann diese Prompts nutzen

- **Vor einer strategischen Entscheidung** – Neuer Markt, neues Produkt, neuer Kanal
- **Am Anfang eines Projekts** – Um den Kontext zu strukturieren bevor man loslegt
- **Im Strategiegespräch** – Als Input für den CEO-Agenten
- **Zur Validierung** – Um eigene Annahmen zu hinterfragen

## Tipp: Kombination

Diese Prompts bauen aufeinander auf:
1. `market-analysis.md` → Verstehe den Markt
2. `competitor-research.md` → Verstehe die Wettbewerber
3. `business-model-canvas.md` → Entwickle dein Modell auf Basis der Erkenntnisse
