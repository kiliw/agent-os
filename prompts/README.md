# Prompt-Bibliothek

Kuratierte, wiederverwendbare Prompts für typische Business-Aufgaben. Jeder Prompt ist in der Praxis getestet, mit klaren Platzhaltern für eigene Daten, und kommt mit Nutzungshinweisen.

## Wie nutzt man die Prompts?

1. Wähle einen Prompt aus der passenden Kategorie
2. Ersetze alle `[PLATZHALTER IN ECKIGEN KLAMMERN]` mit deinen Daten
3. Kopiere den Prompt in dein KI-Tool (Claude, ChatGPT, etc.)
4. Passe den Output an deine Situation an

**Wichtig:** Diese Prompts sind Ausgangspunkte, keine fertigen Lösungen. Das Modell liefert Entwürfe und Strukturen – dein Urteilsvermögen und Kontext-Wissen veredeln sie.

Die meisten Prompts sind mit **Claude Sonnet** oder **GPT-4o** getestet. Für besonders komplexe Analysen empfehlen wir **Claude Opus** oder **o1/o3**. Jede Prompt-Datei enthält eine Empfehlung.
