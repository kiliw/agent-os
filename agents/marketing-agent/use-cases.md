# Marketing-Agent – Use Cases

---

## Use Case 1: Landing-Page-Copy entwickeln

**Input:**
```
Erstelle die Copy für die Landing Page unseres Heimbrauen-Starter-Kits (199€).

Produkt: Komplettset für Einsteiger – alles was man braucht für den ersten Sud (Ausrüstung + Rezept + Anleitung)
Zielgruppe: Männer 30-45, biertrinkendem Hobby suchend, wollen etwas "mit den Händen machen"
Versprechen: "Dein erstes eigenes Bier in 4 Wochen – garantiert"
Differenzierung: Wir inkludieren eine 1h Video-Begleitung und E-Mail-Support – andere Kits lassen einen alleine

Struktur: Hero-Sektion, 3 Hauptargumente, Social Proof Bereich, FAQ (3 Fragen), CTA

Ton: Direkt, ehrlich, ohne Marketing-Floskeln. Duzen.
```

**Erwarteter Output:** Vollständige Landing-Page-Sections mit Headlines, Body Copy, CTA-Text. Je 2 Varianten für Hero-Headline und CTA.

## Use Case 2: E-Mail-Welcome-Sequenz

**Input:**
```
Schreibe eine 5-E-Mail-Welcome-Sequenz für neue Newsletter-Abonnenten.
Abonnenten kommen über: Lead-Magnet "5 Fehler beim ersten Heimbrauen"
Ziel der Sequenz: Vertrauen aufbauen, Marke vorstellen, dann sanft zum Starter-Kit führen

E-Mail 1 (sofort): Lead-Magnet liefern + kurze Vorstellung
E-Mail 2 (Tag 2): Eine echte Tipp-E-Mail (kein Verkauf)
E-Mail 3 (Tag 4): Story – warum ich das Unternehmen gegründet habe
E-Mail 4 (Tag 6): Social Proof – was Kunden sagen
E-Mail 5 (Tag 9): Angebot mit Dringlichkeit

Ton: Persönlich, von Gründer zu Leser, wie eine E-Mail von einem Freund der zufällig Expertise hat.
```

## Use Case 3: Social-Media-Content für eine Woche

**Input:**
```
Erstelle einen Content-Plan für eine Woche Instagram + LinkedIn.

Ziel: Reichweite + Markenbekanntheit aufbauen (kein direkter Verkauf diese Woche)
Thema der Woche: "Warum Heimbrauen 2024 boomt" (basierend auf unserer Research)

Instagram: 3 Posts (Carousel, Reel-Idee, Single Image) + je 5 Stories
LinkedIn: 2 Posts (eher edukativ, Business-Perspektive)

Pro Post: Hook (erste Zeile), Content, Caption, Hashtag-Empfehlung

Ton Instagram: Zugänglich, visuell, community-nah
Ton LinkedIn: Unternehmerisch, Insights-getrieben, professionell aber nicht steif
```

## Use Case 4: Positionierung entwickeln

**Input:**
```
Hilf mir meine Positionierung zu schärfen.
Aktuell: "Premium-Heimbrauen-Ausrüstung für Hobbybrauer"
Problem: Das klingt wie alle anderen auch.

Was uns unterscheidet:
- Wir liefern als einzige im Markt Video-Support und E-Mail-Begleitung
- Unsere Produkte werden von uns selbst getestet bevor sie ins Sortiment kommen
- Kein China-Import-Ramsch – alles von deutschen und österreichischen Herstellern

Zielgruppe: Hobby-enthusiasten die Qualität schätzen und nicht raten wollen

Entwickle 3 alternative Positionierungs-Varianten mit je: Headline, One-liner, 3 Kernbotschaften.
```

## Use Case 5: Anzeigen-Copy für Meta Ads

**Input:**
```
Erstelle 3 Ad-Varianten für Meta (Facebook/Instagram) um Käufer für unser Starter-Kit zu gewinnen.
Budget: 30€/Tag, Ziel: Käufe auf der Landing Page

Zielgruppe: Interessierte an Heimbrauen, Bier, Kochen, DIY (Meta-Targeting)
Preis: 199€
USP: "Alles inklusive + Video-Begleitung"

Pro Variante: Headline, Primary Text (max 125 Zeichen), Link-Beschreibung
Varianten sollen verschiedene Winkel testen: Angst (falsch kaufen), Desire (stolz sein), Social ("alle machen es")
```

## Use Case 6: SEO-Content-Strategie

**Input:**
```
Entwickle eine SEO-Content-Strategie für die nächsten 3 Monate.

Ziel: Organischen Traffic für Käufer-Intent-Keywords aufbauen
Hauptprodukt: Heimbrauen-Starter-Kit (199€)
Bekannte Keywords mit Traffic: "heimbrauen set", "bier brauen zuhause", "brauanlage kaufen"
Content-Ressourcen: 2 Blog-Posts pro Monat (ca. 1500 Wörter)

Liefere: Topic-Cluster-Übersicht, Prioritisierte Liste der ersten 6 Blog-Post-Themen mit Ziel-Keyword und Search-Intent.
```

## Use Case 7: Produkttext für Shopify

**Input:**
```
Schreibe einen überzeugenden Produkttext für unser Flaggschiff-Starter-Kit.

Produkt: "HomeBrewPro Starter-Set Complete"
Preis: 199€
Was enthalten: 8-Liter-Gäreimer, Siphon, Flaschen-Abfüller, Hydrometer, Airlock, Thermometer, Reinigungsmittel, Anfänger-Rezept "Münchner Hell", 1h Begleit-Video, 30 Tage E-Mail-Support
Zielgruppe: Einsteiger die kein Vorwissen haben

Textlänge: 200-300 Wörter
Ton: Ermutigend, klar, keine Fachbegriffe ohne Erklärung
Struktur: Kurze Intro, 3-4 Hauptargumente als Bullets, Abschluss mit CTA
```
