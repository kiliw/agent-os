# CEO-Agent – Use Cases

Konkrete Beispiele wie man den CEO-Agenten einsetzt. Jeder Use Case zeigt den Input den man eingibt und was man zurückbekommt.

---

## Use Case 1: Strategische Prioritäten setzen

**Situation:** Du hast zu viele Ideen und musst entscheiden was als nächstes gebaut/getan wird.

**Input an den Agenten:**
```
Ich habe folgende Projekte/Aufgaben auf dem Tisch:
1. Neue Produktkategorie (Gärungsprodukte) entwickeln – potenziell +30% Umsatz
2. Bestehenden Shop für Mobile optimieren – aktuell 60% Mobile-Traffic, schlechte UX
3. Affiliate-Programm aufbauen – könnte Partner-Kanal werden
4. Kundensupport automatisieren – aktuell 2h/Tag manueller Aufwand
5. Newsletter-Strategie ausbauen – 800 Abonnenten, kaum bespielt

Aktuelles Quartalsziel: Umsatz von 8.000€ auf 15.000€ steigern.
Ressourcen: Ich bin allein, ca. 20h/Woche verfügbar.

Was priorisierst du?
```

**Was der Agent produziert:**
Eine strukturierte Empfehlung mit Priorisierung (Top 2-3 Projekte), Begründung warum die anderen warten können, und einen konkreten 4-Wochen-Plan.

## Use Case 2: Entscheidung zwischen zwei Optionen

**Situation:** Du stehst vor einer konkreten Entweder-oder-Entscheidung.

**Input an den Agenten:**
```
Ich überlege zwischen zwei Vertriebsstrategien:

Option A: Direkt über eigenen Shop, höhere Marge (~40%), aber ich muss selbst Traffic aufbauen
Option B: Über Amazon/Marktplätze, niedrigere Marge (~20%), aber direkter Zugang zu Nachfrage

Aktuell: 0 Umsatz, bin in der Validierungsphase. Budget: 500€/Monat für Marketing.
Ziel: Ersten Proof-of-Concept in 3 Monaten.

Wie würdest du entscheiden?
```

## Use Case 3: OKR-Review und Kurskorrektur

**Situation:** Das Quartal ist halb rum, die Zahlen stimmen nicht. Was tun?

**Input an den Agenten:**
```
Wir sind 6 Wochen ins Q1. Status meiner OKRs:

KR1: 50 zahlende Kunden → Aktuell: 12 (24% des Ziels)
KR2: NPS > 40 → Aktuell: NPS 52 (gut!)
KR3: 3 Vertriebskanäle validiert → Aktuell: 1 (nur organisch SEO)

Was ich weiß:
- Die Kunden die wir haben, lieben das Produkt (NPS bestätigt)
- Der Traffic ist ok (500 unique/Monat), aber die Conversion ist schlecht (~1%)
- Ich habe noch keinen bezahlten Kanal getestet

Empfehlung?
```

## Use Case 4: Pivot-Entscheidung bewerten

**Situation:** Du überlegst ob du das Geschäftsmodell oder den Fokus ändern solltest.

**Input an den Agenten:**
```
Ich habe folgende Signale:
- Meine Starter-Kits (niedrigpreisig, 49€) verkaufen sich schlecht
- Meine Premium-Sets (249€) gehen unerwartet gut (3x Conversion-Rate)
- Mehrere Kunden haben nach Kursen/Workshops gefragt
- Ein B2B-Kunde hat angefragt ob ich für sein Hotel-Restaurant eine Brau-Experience anbiete

Ursprünglicher Plan: Hobbymarkt, niedrigschwellig, breite Masse
Frage: Sollte ich in Richtung Premium + B2B pivotieren?
```

## Use Case 5: Jahresplanung strukturieren

**Situation:** Neues Jahr, du willst strategisch planen statt nur reagieren.

**Input an den Agenten:**
```
Ich will mein nächstes Jahr strukturieren. Hier meine Situation:

Letztes Jahr:
- Umsatz: 85.000€ (Ziel war 100.000€)
- Haupterkenntnis: B2B-Kunden (Gastronomie) sind profitabler als B2C
- Größtes Problem: Zu viele Produkte, zu wenig Fokus

Was ich für nächstes Jahr will:
- Umsatz: 150.000€
- Weniger operative Last, mehr strategische Arbeit
- Eventuell erste(n) Mitarbeiter

Hilf mir, 3 Jahresziele (Objectives) mit je 2-3 Key Results zu formulieren.
```

## Use Case 6: Schnelle Meinung zu einer Idee

**Situation:** Du hast eine Idee und willst einen schnellen Realitätscheck.

**Input an den Agenten:**
```
Idee: Ich will eine Heimbrauen-Community aufbauen (Forum/Discord) um Brand-Loyalität zu stärken und Inhalte von Nutzern zu generieren.

Aufwand: Schätzungsweise 5h/Woche Moderationsaufwand
Kosten: Fast nix (Discord kostenlos)
Erwarteter Nutzen: Mehr Engagement, möglicherweise organische Mundpropaganda

Was denkst du? Lohnt das?
```

## Use Case 7: Vorbereitung auf ein wichtiges Gespräch

**Situation:** Du hast ein Investor-Gespräch, Partnergespräch oder wichtiges Kundengespräch.

**Input an den Agenten:**
```
Morgen habe ich ein Gespräch mit einem potenziellen strategischen Partner:
[PARTNER] ist ein großer Brauerei-Zulieferer mit eigenem Online-Shop (kein Hobbybereich).
Sie haben angefragt ob wir kooperieren wollen.

Mein Interesse: Zugang zu ihrem Kundenstamm, ggf. Bundling unserer Produkte
Ihr vermutliches Interesse: Erschließung Hobbymarkt ohne eigenen Aufbau

Was sind meine Ziele für das Gespräch? Welche Fragen muss ich stellen? Welche roten Linien sollte ich ziehen?
```
